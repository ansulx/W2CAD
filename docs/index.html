W2CAD (Wavelet-to-CAD)

A scalogram-first deep learning workflow for CAD vs NONCAD classification from ABP and PPG signals, using CWT features and ResNet50 transfer learning.

Pipeline overview

From physiological signals to clinical discrimination using wavelet time-frequency representations and CNN transfer learning.

Signal windows

Fixed-length windows from ABP/PPG with missing values replaced by segment median, downsampled for efficient computation.

CWT scalograms

Morlet wavelet on 32 scales, log-power transform, min-max normalization, and 3-channel replication.

ResNet50 transfer learning

ImageNet-pretrained backbone with a binary classifier head, fully fine-tuned on scalogram images.

Research focus

Designed for rigorous evaluation in clinical time-series classification settings.

Evaluation rigor

Subject-level split to prevent leakage, stratified labels, and clinically relevant metrics including AUROC and AUPRC.

Baselines and comparators

Classical feature baselines (Logistic Regression, Random Forest) provide a grounded performance reference.

Reproducible runs

Fixed seeds, repeatable index generation, and a Colab notebook to run the full pipeline on GPU.

Publication readiness

Documented gaps and recommendations for top-tier submissions, including multi-seed evaluation and confidence intervals.

Project resources

Configs

ABP and PPG experiments configured in configs/abp.yaml and configs/ppg.yaml.

Training

scripts/train_resnet50.py fine-tunes ResNet50 with early stopping and validation AUC monitoring.

Reports

scripts/generate_report.py creates a summary report and figures for publication-ready documentation.

Getting started

Quick steps to run the pipeline locally or in Colab.

1

Install dependencies

pip install -r requirements.txt

2

Build subject-level indices

python scripts/make_index.py --config configs/abp.yaml
python scripts/make_index.py --config configs/ppg.yaml

3

Train models

python scripts/train_resnet50.py --config configs/abp.yaml
python scripts/train_resnet50.py --config configs/ppg.yaml

4

Generate reports

python scripts/generate_report.py --configs configs/abp.yaml configs/ppg.yaml --out docs/REPORT.md
